# Bazaaro 🛒
**AI-powered product decision engine for India**

> Don't Compare. Just Decide.

## ✅ What's New (v2)
- Real AI via Groq (free, no credit card)
- Live product analysis across 6 platforms
- Trust Score & fake review detection
- Deep Analysis toggle per product
- Buy links to best platform
- Works 100% on GitHub Pages (no server)

## 🚀 Deploy in 2 Steps
1. Upload `index.html` to your GitHub repo
2. Enable GitHub Pages → Done!

## 🔑 API Key
- Go to https://console.groq.com → Create free account → Copy API key
- Open the site → click "API Key" → paste key → Save
- Key is stored in your browser only, never sent anywhere else

## 💡 Tech Stack
- Pure HTML/CSS/JS (zero dependencies)
- Groq API (Llama 3.3 70B, free tier)
- localStorage for key persistence
- No backend, no Node.js needed
